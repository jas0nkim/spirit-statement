== Description ==

Real time stats from StatPress for your Wordpress Dashboard - Lite-Version - much smaller and faster. Based on the original StatPress Widget by <a href="http://www.irisco.it">Daniele Lippi</a>.

**Features**

* much smaller and faster - only the key facts
* no slowing, design-burning graph
* fits also on 3-column view
* just install and place the widget on your dashboard

== Installation ==

You can use the built in installer and upgrader, or you can install the plugin manually.

**Installation via Wordpress**

1. Go to the menu 'Plugins' -> 'Install' and search for 'StatPress Dashboard Widget Lite'
2. Click 'install'

**Manual Installation**

1. Upload folder 'statpress-dashboard-widget-lite' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
